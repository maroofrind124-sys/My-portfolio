## 🚀 Projects

### 1️⃣ [Project 1 — Boston Housing Price Prediction](./ml-projects/project1)
**Type:** Regression | **Model:** Linear Regression
Predicts median house prices using the Boston Housing dataset, with detailed EDA and feature engineering (log transforms, polynomial features, custom ratios).
- **R² Score:** 0.77 | **RMSE:** 4.14

### 2️⃣ [Project 2 — Titanic Survival Prediction](./ml-projects/project2)
**Type:** Classification | **Model:** Logistic Regression
Predicts passenger survival on the Titanic based on class, sex, age, fare, and other features.
- **Accuracy:** 80.3% | **AUC Score:** 0.797
