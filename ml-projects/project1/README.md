🏡 Project: Boston Housing Price Prediction using Linear Regression

I recently completed a Machine Learning project where I predicted house prices using the Boston Housing dataset and a Linear Regression model.

🔍 What I did:
✅ Exploratory Data Analysis (distributions, correlations, outliers)
✅ Data Cleaning (handled duplicates and missing values)
✅ Feature Engineering (log transforms, polynomial features, custom ratios, age categories)
✅ Multicollinearity check using a correlation heatmap
✅ Feature scaling with StandardScaler
✅ Model training & evaluation

📊 Results:
• R² Score: 0.77
• Adjusted R²: 0.73
• MAE: 2.79
• RMSE: 4.14

This project taught me that real-world data is messy, and good predictions need more than just fitting a model — proper EDA and feature engineering are equally important.
